# godot-rs-config

A command line tool to create a godot extension library.

## Install
``` bash
  $ cargo install godot-rs-config
```

## Generate Project
``` bash
  $ godot-config new <lib_name>
```


## Generate Godot class struct
``` bash
  $ godot-config class <className> <baseNode>

```
